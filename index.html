A console-based C++17 application that calculates Body Mass Index, classifies results using WHO guidelines, and stores user records in a local PSV database.

📐 C++17 🏥 WHO Classification 💾 PSV Persistence

About the Project

What It Does & Why

This system automates a common health metric workflow: collect height and weight, compute BMI, assign a standardized category with health guidance, and optionally save results for later review.

Problem Statement

Manual BMI calculation and record-keeping are error-prone and inconsistent. This system automates the formula, applies standardized classification rules, validates user input, and persists records so users can track measurements over time without a full database server.

Target Users

Individuals — monitoring personal health metrics

Students — learning object-oriented design in C++

Educators — demonstrating introductory programming concepts

BMI Formula

BMI = weight (kg) / [height (m)]²

Reading Records — Non‑owning Pointers

📄 src/file_manager.cpp raw pointer access
1// Returns a vector of non-owning raw pointers to the stored records.
2std::vector<const User *> FileManager::read_all() const {
3 std::vector<const User *> ptrs;
4 ptrs.reserve(records.size());
5
6 // Iterate over the unique_ptrs, but only give away raw pointers.
7 for (const auto &u : records) {
8 ptrs.push_back(u.get()); // get() returns the raw pointer (non-owning)
9 }
10
11 return ptrs;
12}

Note: u.get() gives a raw pointer without transferring ownership. The unique_ptr still owns and manages the heap memory. This pattern is safe because the caller does not delete the pointer.

Source Code — PSV Serialization (User)

📄 src/user.cpp to_psv / from_psv
1// Serialize User object to pipe-separated string for file storage
2std::string User::to_psv() const {
3 std::ostringstream oss; // Use string stream for efficient concatenation
4 oss << id << "|" << name << "|" << gender << "|" << age << "|" << height
5 << "|" << weight << "|" << bmi << "|" << category << "|" << advice << "|"
6 << risk; // Pipe '|' separates each field
7 return oss.str();
8}
9
10// Parse a pipe-separated line back into a User object
11User User::from_psv(const std::string &psvLine) {
12 std::istringstream iss(psvLine);
13 std::string token; // Temporary storage for each field
14
15 // Default values in case parsing fails
16 int id = 0;
17 std::string name, gender, category, advice, risk;
18 int age = 0;
19 double height = 0.0, weight = 0.0, bmi = 0.0;
20
21 try {
22 // Parse each field in order, converting types where needed
23 if (std::getline(iss, token, '|')) {
24 id = std::stoi(token); // Convert string to int
25 }
26 std::getline(iss, name, '|'); // Name stays as string
27 std::getline(iss, gender, '|'); // Gender stays as string
28 if (std::getline(iss, token, '|')) {
29 age = std::stoi(token);
30 }
31 if (std::getline(iss, token, '|')) {
32 height = std::stod(token); // Convert string to double
33 }
34 if (std::getline(iss, token, '|')) {
35 weight = std::stod(token);
36 }
37 if (std::getline(iss, token, '|')) {
38 bmi = std::stod(token);
39 }
40 std::getline(iss, category, '|'); // These remain strings
41 std::getline(iss, advice, '|');
42 std::getline(iss, risk); // Last field (no trailing delimiter)
43 } catch (const std::exception &e) {
44 // Log error but continue – skip malformed record instead of crashing
45 std::cerr << "Warning: skipping malformed PSV record: " << e.what() << "\n";
46 }
47
48 // Construct and return a new User object with parsed values
49 return User(id, name, gender, age, height, weight, bmi, category, advice, risk);
50}

Pattern: to_psv() serializes a record into a pipe‑separated line using std::ostringstream; from_psv() parses it back with error handling. This keeps file I/O logic clean.

Source Code — Automatic Backup System

📄 src/file_manager.cpp backup()
1void FileManager::backup() {
2 if (!fs::exists(db_file_path)) {
3 return;
4 }
5
6 const std::string backup_folder = db_folder + "/backup";
7 if (!fs::exists(backup_folder)) {
8 fs::create_directories(backup_folder);
9 }
10
11 const auto now = std::chrono::system_clock::now();
12 const std::time_t time = std::chrono::system_clock::to_time_t(now);
13 std::tm localTime{};
14#ifdef _WIN32
15 localtime_s(&localTime, &time);
16#else
17 localtime_r(&time, &localTime);
18#endif
19
20 std::ostringstream timestamp;
21 timestamp << std::put_time(&localTime, "%Y-%m-%d_%H-%M-%S");
22 const std::string backup_path =
23 backup_folder + "/records_" + timestamp.str() + ".psv";
24
25 try {
26 fs::copy_file(db_file_path, backup_path,
27 fs::copy_options::overwrite_existing);
28 } catch (const fs::filesystem_error &) {
29 return;
30 }
31
32 std::vector<fs::directory_entry> backupFiles;
33 for (const auto &entry : fs::directory_iterator(backup_folder)) {
34 if (entry.is_regular_file() && entry.path().extension() == ".psv") {
35 backupFiles.push_back(entry);
36 }
37 }
38
39 if (backupFiles.size() <= 3) {
40 return;
41 }
42
43 std::sort(backupFiles.begin(), backupFiles.end(),
44 [](const fs::directory_entry &a, const fs::directory_entry &b) {
45 return fs::last_write_time(a) < fs::last_write_time(b);
46 });
47
48 while (backupFiles.size() > 3) {
49 fs::remove(backupFiles.front().path());
50 backupFiles.erase(backupFiles.begin());
51 }
52}

Note: backup() creates a timestamped copy after every create/update, and automatically deletes old backups (keeps only the 3 newest). This prevents data loss.
